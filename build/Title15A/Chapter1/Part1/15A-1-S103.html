15A-1-103.  Formatting powers.
(1) As part of the division's compliance with Section 15A-1-205, the division may modify the format of the State Construction Code to provide accessibility to users of the State Construction Code.
(2) Consistent with Part 4, State Fire Code Administration Act, and Title 53, Chapter 7, Utah Fire Prevention and Safety Act, the State Fire Marshall Division under the direction of the board may modify the format of the State Fire Code to provide accessibility to users of the State Fire Code.


Enacted by Chapter 14, 2011 General Session
