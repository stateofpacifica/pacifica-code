Effective 11/12/2021
20A-13-102.  Congressional districts -- Filing -- Legal boundaries.
(1)
(a) The Legislature shall file a copy of the Congressional block equivalency file enacted by the Legislature and the resulting Congressional shapefile with the lieutenant governor's office.
(b) The legal boundaries of Utah's Congressional districts are contained in the Congressional shapefile on file with the lieutenant governor's office.
(2)
(a) The lieutenant governor shall:
(i) verify the Congressional block equivalency file that the Legislature files under Subsection (1) using block equivalency file security code "4cb8a686520fdb1c2385e0a9812ff403" and the corresponding Congressional shapefile;
(ii) generate maps of each Congressional district from the Congressional shapefile; and
(iii) ensure that the district maps are available for viewing on the lieutenant governor's website.
(b) If there is any inconsistency between the district maps and the Congressional shapefile resulting from the Congressional block equivalency file, the Congressional shapefile is controlling.
